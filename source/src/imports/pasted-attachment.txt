OPERATIONS PAGE REQUIREMENTS (Super Simple):

1. 📁 CASE FOLDER MANAGER:
   - Big button: "➕ New Case Folder."
   - Form: Client Name* | Phone* | Destination* → Click "Create"
   - Auto-generates: Folder ID (EMR-2024-XXX) + assigns to operator or dropdown to assign to agent
   - Visual: Folder icon + client name + assigned person badge
   - Search bar: Type name/phone → instant filter
   - He must check all the documents. If anything is missing, he has to arrange the full document offline and online, both Althought except for the Master agent, he will be the only person to see and approve payments and documents and download them, but cannot back up for compliance reasons 

2. ✅ STATUS CONFIRMATION DASHBOARD:
   - Simple list view (NOT complex table):
     ```
     [📁 EMR-001] Ahmed Khan → 🟡 Medical → [✓ Confirm] [⏳ Pending]
     [📁 EMR-002] Hassan Ali → 🔵 Documents → [✓ Confirm] [⏳ Pending]
     ```
   - One-click confirm: Click [✓] → Status updates + notification to Admin
   - Color-coded: 🟢 Done | 🟡 In Progress | 🔴 Overdue
   - Filter: Show only "Pending Confirmations"

3. 📅 APPOINTMENT MANAGER (Auto + Manual):
   - Auto-Generate Button: "🔄 Generate Next Appointments"
     * System suggests: Medical follow-ups, Protector 8 AM slots, Payment reminders
     * Based on case stage + deadlines
   - Manual Add Button: "➕ Add Appointment"
     * Simple form: Client* | Type* (Medical/Protector/Payment) | Date/Time* | Notes
     * Large date picker + time slots (8 AM, 10 AM, 2 PM presets)
   - Calendar View: Simple monthly grid with colored dots per appointment type
   - Today's List: Big cards showing: Client | Time | Type | [✓ Done] [📞 Remind]

4. 🎯 AGENT SUPPORT MODE (Backup for Agents):
   - Toggle: "Show Pending Agent Tasks"
   - List: Tasks missed by agents (overdue documents, unconfirmed payments, etc.)
   - One-click take over: [🔄 I'll Do This] → Task assigned to operator
   - Visual cue: "Agent Farhan missed: Upload CNIC for EMR-001" → [Fix Now]

5. 👥 ATTENDANCE MARKER (Compulsory):
   - Big buttons for each staff member:
     ```
     [👤 Agent Farhan]  [✅ Present] [⏰ Late] [❌ Absent]
     [👤 Agent Ayesha]  [✅ Present] [⏰ Late] [❌ Absent]
     [👤 Operator You]  [✅ Present] [⏰ Late] [❌ Absent]
     ```
   - Click status → Auto-logs time + location (if mobile)
   - Monthly view: Simple calendar with colored dots (Green=Present, Yellow=Late, Red=Absent)
   - Export: "Download Attendance" → CSV (for payroll)

6. 🏢 OFFICE VISIT LOGGER:
   - Big button: "➕ Log Office Visit"
   - Simple form: Client Name* | Phone* | Purpose* (Consultation/Document/Payment) | Met With* (Dropdown: Expert/Agent/Operator) | Notes
   - Auto-saves with timestamp
   - Today's Visits: Simple list with client name + time + purpose

7. 💰 PAYMENT DOCUMENTATION (Simple):
   - Big button: "➕ Record Payment"
   - Form: Client* | Amount* | Method* (Cash/EasyPaisa/JazzCash/Bank) | Receipt Photo [📸 Upload]
   - Auto-generates: Receipt # + timestamp
   - List view: Date | Client | Amount | Method | [📄 View Receipt]
   - Filter: Today | This Week | This Month

DESIGN RULES (Operator-Friendly):

- Visual cues over text: Icons + colors > long descriptions
