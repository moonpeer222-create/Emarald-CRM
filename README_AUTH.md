## 🧪 How to Get Started

### Step 1: Create Your First Account
1. Visit: https://wasi-app-1.web.app
2. Click "Create Account"
3. Fill in your details
4. Select "Master Admin" role
5. Click "Create Account"
6. You're in! 🎉

### Step 2: Add Team Members
- Each team member signs up at the same URL
- They select their appropriate role
- They create their own password
- Secure access for everyone

### Step 3: Start Using the CRM
- All features now available to authenticated users
- No more demo credentials or hardcoded accounts
- Professional authentication
