UNIVERSAL CRM - DEPLOYMENT PACKAGE
====================================

✅ WHAT'S NEW IN THIS BUILD:
1. Generic CRM system - NO visa-specific branding
2. Configurable for ANY business type:
   - Sales CRM
   - Real Estate  
   - Recruitment
   - Project Management
   - Custom

3. Mobile-responsive design:

   - Sign In / Sign Up / Forgot Password
   - Firebase Authentication
   - Role-based Access Control
   - Password Reset via Email
   - User Profile Management

📦 TO DEPLOY TO FIREBASE:
==========================
1. Install Firebase CLI:
   npm install -g firebase-tools

2. Login to Firebase:
   firebase login

3. Navigate to this folder and deploy:
   firebase deploy --project wasi-app-1

📋 AUTHENTICATION SETUP:
========================
After deployment, follow these steps to set up users:

1. Visit: https://wasi-app-1.web.app
2. Click "Create Account" to add users or sign up
3. Choose your role:
   - Master Admin (Full system access)
   - Admin (Administrative functions)
   - Agent (Customer management)
4. Set a strong password (minimum 6 characters)
5. Use "Forgot Password?" for password reset

For detailed authentication setup, see: AUTH_SETUP.md
